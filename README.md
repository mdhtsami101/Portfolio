# Medhat Samy | Flutter Developer Portfolio

A premium, high-performance portfolio website built with HTML, CSS, and Javascript. 

## Features
- **Nebula Design**: Animated gradients, floating glassmorphism, and editorial typography.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Custom Cursor**: Interactive magnetic cursor for an immersive experience.
- **Vercel Optimized**: Includes `vercel.json` for seamless deployment.

## Deployment
This project is ready to be deployed on [Vercel](https://vercel.com). Simply connect your GitHub repository and it will automatically detect the static build.
